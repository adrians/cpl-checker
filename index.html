Grammar editor

  • The grammar must be called "Demo". If it's of the "lexer grammar" type it won't work.
  • Press the "compile lexer" button and wait 7-8 seconds for the server to compile the lexer and parser.
  • The "grammar Demo has no non-fragment rules" error can be ignored.
  • If the lexer and the parser were generated correctly, editing and running the lexer should generate output.
